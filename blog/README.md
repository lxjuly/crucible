# Crucible Blog

This directory contains public writing for the Crucible project.

The goal is to write alongside the codebase, so every essay can point to concrete commits, demos, traces, and design decisions.

## Writing principles

- Start from a concrete failure mode.
- Explain the system abstraction.
- Show the minimal demo.
- Include tradeoffs and limitations.
- Link ideas back to code.
